# PrivacyPreservingML
Repository for the 2025 Integrated Masters Project by Cedric Bohni and Konstantin Wehmeyer. 

This project contains notebooks to test different privacy-preserving machine learning techniques as well as a demo application showcasing secure inference using functional encryption.
Inner-product functional encryption was used for the functional encryption approach.

python -m src.Demo.server
